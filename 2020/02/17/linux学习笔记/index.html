硬件设备文件名在/dev
SATA,USB硬盘 /dev/sd[a-p]
ex:/dev/sda1 a:第一块硬盘 1:第一个分区
假设你的PC上有两个SATA磁盘和一个USB磁盘,上面有6个插槽,该怎么命名?
注:命名只与安装顺序有关,与插哪个口无关,第一个就/etc/sda
USB磁盘开机完成后才被系统识别,所以只能是最后一个
每一个硬盘最多只有4个分区
df查看磁盘剩余空间的命令
df -h查看硬盘使用情况
df -h -T并查看类型
关机重启
shutdown是最常用也是最安全的关机和重启命令，它会在关机之前调用fsck检查磁盘，其中-h和-r是最常用的参数：
-h：停止系统服务并关机
-r： 停止系统服务后重启
举个例子：
shutdown -h now –立即关机
shutdown -h 10:53 –到10:53关机，如果该时间小于当前时间，则到隔天
shutdown -h +10 –10分钟后自动关机
shutdown -r now –立即重启
shutdown -r +30 ‘The System Will Reboot in 30 Mins’ –30分钟后重启并并发送通知给其它在线用户

ls -a显示隐藏目录 -l列举详细信息 -ld列出当前目录
ls -l
第一位表示文件类型 d目录文件 l软连接文件 -普通文件
2-4位文件所有者权限 5-7位表示同组用户的权限 8-10位表示其他用户的权限
r读 w写 x执行 -无权限
文件处理命令
cd - 进入上次目录
mkdir -p xxx 同时建立目录名的上层目录
rmdir xxx 注意只能删除空目录 -p 顺带删除父目录
rm -r 删除目录及其下的文件和子目录
rm -f 强制删除
rm -i 一一确认再删除
rm -rf /后果严重 删除根目录下的所有文件
ln 生成链接文件 默认建立硬链接
-s 软链接
